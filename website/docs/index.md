# The rkt Programming Guide

Welcome to rkt!

This is the official guide for rkt v1.0.1. It is designed to serve as a
starting point to writing web applications with rkt and Rust.

The guide is also designed to be a reference for experienced Rocket developers.

For purely technical documentation with examples, see the
[API documentation](https://docs.rs/rkt/latest/rkt).

## Why rkt

rkt is a fork of the Rocket framework. The intention is to provide more timely updates,
stabilize the API and keep the development going.

Shout out Sergio at. al. who built such a brilliant framework!

## Getting Help

If you need help, find a bug or can use a new feature, please reach out on the [GitHub
Repository](https://github.com/rustfoo/rkt).
